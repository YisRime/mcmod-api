MCMod中文网 API

用于获取MCMod中文网数据的API服务

API 端点

获取模组详情

GET /api/mod?id={modId}

返回指定ID的模组详细信息

参数类型说明
id数字模组的唯一标识符

示例: /api/mod?id=2

获取物品详情

GET /api/item?id={itemId}

返回指定ID的物品详细信息

参数类型说明
id数字物品的唯一标识符

示例: /api/item?id=1234

获取模组包详情

GET /api/modpack?id={packId}

返回指定ID的模组包详细信息

参数类型说明
id数字模组包的唯一标识符

示例: /api/modpack?id=567

获取教程详情

GET /api/course?id={courseId}

返回指定ID的教程详细信息

参数类型说明
id数字教程的唯一标识符

示例: /api/course?id=789

获取服务器详情

GET /api/server?id={serverId}

返回指定ID的服务器详细信息

参数类型说明
id数字服务器的唯一标识符

示例: /api/server?id=123

搜索内容

GET /api/search?q={查询词}

使用关键词搜索所有类型的内容

参数类型说明
q字符串搜索关键词

示例: /api/search?q=机械

搜索服务器

GET /api/search/server?q={查询词}&page={页码}

搜索服务器，支持分页

参数类型说明
q字符串搜索关键词
page数字页码，默认为1

示例: /api/search/server?q=生存&page=1

获取模组列表

GET /api/list?category={分类}&page={页码}

获取指定分类的模组列表，支持分页

参数类型说明
category字符串分类名称（可选）
page数字页码，默认为1

示例:

  • 所有模组：/api/list?page=1
  • 特定分类：/api/list?category=magic&page=1

注意事项

  • 所有API请求都支持CORS
  • 请求有10秒超时限制
  • 数据来源于mcmod.cn，请遵守原站的使用条款
